# Image2 设计提示词

以下提示词用于生成原创素材，不要输入任何第三方游戏截图，不要要求复刻现有游戏界面。

## 1. 游戏背景

```text
Create an original cute mobile game background, vertical 9:16, cozy farm pond scene, soft green grass, small pond, warm sunlight, rounded cartoon style, clean composition, no text, no logo, pastel colors, suitable for a casual puzzle game UI.
```

## 2. 大鹅角色

```text
Create an original cute white goose character for a casual mobile puzzle game, orange beak and feet, playful expression, rounded cartoon sticker style, soft shadow, transparent background, no text, no logo, front view.
```

## 3. 物品图标套组

```text
Create a consistent set of cute cartoon item icons for a casual match puzzle game: corn, carrot, small fish, apple, bread, leaf, bucket, all in rounded sticker style, soft shadow, transparent background, no text, no logo, consistent lighting and angle.
```

## 4. 开始弹窗插图

```text
Create an original cute illustration for a mobile game start popup: a white goose peeking from grass near a pond, playful and friendly, rounded cartoon style, pastel colors, transparent background, no text, no logo.
```

## 5. 成功弹窗插图

```text
Create an original cute victory illustration for a casual mobile game: a happy white goose sitting in a small basket with stars and confetti, warm pastel colors, rounded cartoon style, transparent background, no text, no logo.
```

## 6. UI 风格参考提示词

```text
Design an original vertical mobile casual puzzle game UI, cute farm pond theme, top score and timer cards, center pile of rounded item stickers, bottom seven-slot tray, three tool buttons, pastel colors, soft shadows, clean layout, no text, no logo, no copied game interface.
```
